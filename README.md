# Windows-mp3-player
a mp3 player for windows.<br><hr>
media control keys are
<ul>
<li>fn+f6 is the previous key</li>
<li>fn+f7 is the play pause key</li>
<li>fn+f8 is the next key</li>
</ul>
<hr>
